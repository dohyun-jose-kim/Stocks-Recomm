# 프로젝트 개요

## 한 줄 요약
예측시장(Polymarket, Kalshi)을 따라가면서, 군중이 지금 어떤 이슈를 어떻게 보고 있는지를 추적하는 개인 프로젝트.

## 왜 하는가
**이슈 파악이 재미있다.** 뉴스를 일일이 따라가지 않아도, 사람들이 돈을 걸고 있는 사건들의 확률 추이를 보면 "지금 세상에서 무슨 일이 중요해지고 있는지"가 보인다. Fed 금리, CPI, 대선, 지정학 이슈 — 군중 판단이 실시간으로 숫자로 찍힌다.

거창한 알파를 발굴한다거나 자동매매를 만들겠다는 게 아니다. **확률 곡선을 읽으면서 이슈를 빨리/재미있게 따라가는 도구.** 부수적으로 시계열이 쌓이면 나중에 뭐든 분석해볼 수 있다.

## 무엇을 하는가
- **수집**: Polymarket + Kalshi에서 관심 마켓의 내재확률을 주기적으로 스냅샷
- **저장**: SQLite에 시계열로 누적
- **탐색**: 확률 추이 차트, 급변 감지, 두 플랫폼 간 괴리 비교

## 무엇이 아닌가
- 자동매매 아니다.
- 퀀트 팩터 모델, CAPM, 파마-프렌치 — 아니다. (이전 계획에서 의도적으로 뺌)
- 뉴스/가격/매크로/공시 종합 파이프라인 아니다. 그건 예측시장 데이터가 쌓이고 진짜 필요해지면 그때 검토.
- "내일 뭐 오른다" 예측기 아니다.

## 스코프 축소 메모 (2026-05-21)
이전에는 한국 뉴스 + 미국 예측시장 + 가격 + 매크로 + 이벤트 + 센티먼트/토픽/교차분석까지 다루는 큰 그림이었음. 퀀트급 야심은 현실적으로 어렵다고 판단해 **예측시장 한 축만 남기고 전부 잘라냄**. 이전 계획은 `archive/2026-04-19_quant-scope/`에 그대로 보존.

## 기술 스택
- **언어**: Python only
- **저장**: SQLite
- **데이터 소스**: Polymarket Gamma/CLOB API, Kalshi Trade API (둘 다 키 불필요로 시작)
- **실행 환경**: macOS 로컬, launchd 스케줄링
- **시각화**: matplotlib/plotly (필요해질 때)

## 프로젝트 구조
```
plans/        계획 문서 (4개 파일)
src/          소스 코드 (빈 골격, 구현 시 채움)
  collector/  Polymarket, Kalshi 수집기
  storage/    DB 스키마/접근
  explore/    탐색/시각화
  config/     마켓 화이트리스트
data/         SQLite DB + 로그 (gitignore)
  db/
  logs/
notebooks/    탐색용 노트북
archive/      이전 광범위 계획 (참고용, 날짜 prefix)
```
